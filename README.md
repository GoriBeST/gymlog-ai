# FORMA

> AI-powered gym log — トレーニング・食事・体重を一元管理するウェブアプリ

単一の HTML ファイルで動作します。サーバー不要、ブラウザで開くだけで使えます。

> [!NOTE]
> 現在 **beta** 版です。UIや機能は頻繁に更新・修正が入ります。最新の `forma.html` に差し替えることでアップデートできます。データは `localStorage` に保存されるため、ファイルを差し替えても記録は引き継がれます。

## セットアップ

1. `forma.html` をダウンロード
2. ブラウザで開く（iPhone の場合は Safari 推奨）
3. 右上の **API KEY** をタップして Anthropic API キーを入力・保存
4. 完了 — AI 機能が全て使えるようになります

> **API キーの取得**：[console.anthropic.com](https://console.anthropic.com) でアカウント作成後、API Keys ページから発行できます。

## 機能

### 💪 トレーニング

| 機能 | 詳細 |
|---|---|
| 手動入力 | 部位別 / PPL / 上下半身の3つの分割法に対応 |
| ノート撮影 | 手書きノートを撮影するとAIがセットを読み取って自動登録 |
| 種目リスト | 130種目以上をプリセット収録（胸・背中・肩・腕・脚・腹・有酸素） |
| カスタム種目 | 直接入力 → AIが部位・器具を自動分類して次回から一覧に表示 |
| 有酸素対応 | マシン（分×km/h）/ 屋外（分×km）/ HIIT（セット×秒）でフィールドが切り替わる |
| トレッドミル | 傾斜（%）フィールドを追加で記録可能 |
| 前回提案 | 前回の重量・rep を自動表示してセット入力を補助 |
| 記録日変更 | 過去の日付で記録することも可能 |

### ⏱ インターバルタイマー

- プリセット：1:00 / 1:30 / 2:00 / 3:00
- 秒数を自由入力して設定可能
- 終了時：バイブ + 三音ビープ + カード点滅
- ブラウザ通知対応（Android / デスクトップ。iOS は ホーム画面に追加後）

### 🍽 食事

- 食事の写真を撮影するとAIがカロリー・PFCを自動解析
- 食事名・カロリー・タンパク質・脂質・炭水化物を記録
- 日々のPFCをバーチャートで可視化

### 🎯 ゴール

- 現在体重・目標体重・達成期限を設定
- BMR / TDEE から目標PFCを自動計算（身長・体重・年齢・性別・活動量）
- AIがプロフィールを見てパーソナライズされたアドバイスを生成

### 📊 記録

- カレンダーでトレーニング・食事を一覧確認
- 体重推移グラフ（折れ線）
- 種目別最大重量グラフ
- PFC推移グラフ（週間バーチャート）

### 📤 投稿

- その日のトレーニング記録をもとにAIがInstagram用の投稿文を生成

## 操作

| 操作 | 動作 |
|---|---|
| タブをタップ | 画面切り替え |
| 左右スワイプ | 画面切り替え（入力中は無効） |
| 日付をタップ | カレンダーピッカーで記録日を変更 |

## 技術仕様

- **単一ファイル**：HTML / CSS / JavaScript をすべて `forma.html` に同梱
- **ストレージ**：`localStorage` にデータを保存（端末内のみ、クラウド同期なし）
- **AI**：Anthropic API（claude-haiku-4-5 / claude-sonnet-4-6）
- **チャート**：Chart.js 4.4.1
- **フォント**：DM Mono、Noto Sans JP（Google Fonts）

---

## 注意事項

- データは端末のブラウザに保存されます。ブラウザのデータを消去すると記録が失われます
- API キーはブラウザの `localStorage` に保存されます。共有端末でのご使用はご注意ください
- iOS Safari でブラウザ通知を受け取るには、ホーム画面に追加（PWA化）が必要です
- オフライン時はAI機能が使えませんが、手動入力・記録閲覧は可能です
